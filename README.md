# holbertonschool-web_back_end

## Description

This repository contains backend development projects from the **Holberton School Web Back End curriculum**. It focuses on building core backend skills using JavaScript, Python, databases, and APIs.

## Technologies

* JavaScript (ES6)
* Node.js
* Python
* SQL & NoSQL
* Git

## Structure

Each folder represents a separate backend project or concept.

## Usage

Clone the repository and follow the instructions inside each project directory.

```bash
git clone https://github.com/Bryanjose001/holbertonschool-web_back_end.git
```
