# ES6_data_manipulation

## Description

This directory contains exercises focused on **data manipulation using ES6 (JavaScript)** as part of the Holberton School backend curriculum.

## Topics

* Array methods (`map`, `filter`, `reduce`)
* Object manipulation
* Destructuring
* Spread and rest operators
* Sets and Maps

## Usage

Run the JavaScript files with Node.js:

```bash
node filename.js
```
