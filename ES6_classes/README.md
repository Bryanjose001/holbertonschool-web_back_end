
# ES6_classes

## Description
This directory contains exercises focused on **ES6 classes** as part of the Holberton School backend curriculum. ES6 classes provide a modern syntax for creating objects and working with inheritance in JavaScript.

## Topics
- Class declaration and instantiation  
- Constructors and methods  
- Inheritance with `extends`  
- Static methods and properties  

## Usage
Run the JavaScript files using Node.js:

```bash
node filename.js
```
