# NoSQL

## Description
This directory contains exercises focused on **NoSQL databases** as part of the Holberton School backend curriculum. NoSQL systems are non-relational databases that store and query data differently than traditional SQL databases, often using JSON-like documents. :contentReference[oaicite:0]{index=0}

## Topics
- NoSQL basics and concepts  
- CRUD operations  
- Working with document-oriented data  
- Querying and filtering data  

## Usage
Refer to the individual files and follow instructions to run or test the implementations.
