# ES6_basic

## Description

This directory contains exercises and projects focused on **ES6 (ECMAScript 2015) basics**, completed as part of the Holberton School backend curriculum.

## Topics Covered

* let and const
* Arrow functions
* Default parameters
* Rest and spread operators
* Template literals
* Modules

## Usage

Run the JavaScript files using Node.js:

```bash
node filename.js
```
