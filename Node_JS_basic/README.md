# Node_JS_basic

## Description
This directory contains exercises focused on **Node.js basics**, completed as part of the Holberton School backend curriculum. It covers fundamental concepts for building and running JavaScript applications on the server.

## Topics
- Node.js environment setup  
- Modules and `require`  
- File system operations  
- Simple HTTP servers  

## Usage
Run the JavaScript files using Node.js:

```bash
node filename.js
